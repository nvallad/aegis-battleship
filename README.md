This project uses a Python virtual environment (venv) and a single command to install all required dependencies.

Requirements

- Python 3.10+
- pip (comes with Python)
- Pygame==2.1.3

Quick Setup


Follow these steps to set up the project on any machine.

1. Create a virtual environment

```
python -m venv venv
```

2. Activate the virtual environment

Windows PowerShell

```powershell
.\venv\Scripts\Activate.ps1
```

Windows CMD

```bat
venv\Scripts\activate.bat
```

Mac/Linux

```bash
source venv/bin/activate
```

3. Install all dependencies (ONE COMMAND)

Once your venv is active, run:

```bash
pip install -r requirements.txt
```

This installs everything the project needs.
